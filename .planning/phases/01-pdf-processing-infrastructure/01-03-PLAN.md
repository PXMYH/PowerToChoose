  <action>
1. Create api/services/pdf_processor.py with:

   - `def classify_pdf(file_path: str) -> PDFClassification`:
     - Open PDF with pdfplumber.open(file_path).
     - Extract text from all pages, concatenate.
     - If total text length < settings.SCANNED_PDF_TEXT_THRESHOLD: PDFType.scanned.
     - Else: PDFType.text_based.
     - Return PDFClassification with pdf_type, text_length, file_path.
     - Handle pdfplumber exceptions gracefully (return PDFType.unknown on parse errors).

   - `def extract_text(file_path: str) -> TextExtractionResult`:
     - Open PDF with pdfplumber.open(file_path).
     - Extract text from each page, join with "\n\n" (double newline as page separator).
     - Strip excessive whitespace per page but preserve structure.
     - Classify inline (reuse threshold logic, don't call classify_pdf to avoid opening twice).
     - Return TextExtractionResult with text, page_count, file_path, pdf_type.

   Note: These are synchronous functions (pdfplumber is sync). The caller (background task in Plan 04) will run them in a thread pool via asyncio.to_thread().
