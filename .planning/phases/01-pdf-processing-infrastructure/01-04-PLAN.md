```sql
CREATE TABLE IF NOT EXISTS jobs (
    id TEXT PRIMARY KEY,
    plan_id TEXT NOT NULL,
    efl_url TEXT,
    status TEXT NOT NULL DEFAULT 'queued',
    pdf_type TEXT,
    error TEXT,
    created_at TEXT NOT NULL DEFAULT (datetime('now')),
    updated_at TEXT NOT NULL DEFAULT (datetime('now'))
);

CREATE INDEX IF NOT EXISTS idx_jobs_plan_id ON jobs(plan_id);
CREATE INDEX IF NOT EXISTS idx_jobs_status ON jobs(status);
```

3. Create api/database/connection.py with:
   - `async def init_db()`:
     - Create data/ directory if not exists.
     - Open aiosqlite connection to settings.DATABASE_PATH.
     - Enable WAL mode: `PRAGMA journal_mode=WAL`.
     - Execute schema.sql to create tables.
     - Close connection.
   - `async def get_db() -> aiosqlite.Connection`:
     - Open and return aiosqlite connection (caller responsible for closing).
     - Set row_factory = aiosqlite.Row for dict-like access.
   - `async def create_job(plan_id: str, efl_url: str) -> str`:
     - Generate UUID4 as job id.
     - INSERT into jobs table with status='queued'.
     - Return job id.
   - `async def update_job_status(job_id: str, status: str, error: str | None = None, pdf_type: str | None = None)`:
     - UPDATE jobs SET status=?, error=?, pdf_type=?, updated_at=datetime('now') WHERE id=?
   - `async def get_job(job_id: str) -> dict | None`:
     - SELECT * FROM jobs WHERE id=?
     - Return dict or None.

4. Update api/main.py startup event to call init_db():
   - Add `@app.on_event("startup")` or lifespan handler that calls `await init_db()`.
   - Keep the existing cache dir creation.

   - `async def process_efl_task(job_id: str, efl_url: str)`:
     - This is the background task entry point. It orchestrates the full pipeline.
     - Stage 1 — Download:
       - await update_job_status(job_id, "downloading")
       - result = await download_pdf(efl_url)  (import from services.downloader)
       - If not result.success: await update_job_status(job_id, "failed", error=result.error); return
     - Stage 2 — Extract + Classify:
       - await update_job_status(job_id, "extracting")
       - extraction = await asyncio.to_thread(extract_text, result.file_path)  (sync function, run in thread)
       - await update_job_status(job_id, status="extracting", pdf_type=extraction.pdf_type.value)
       - If extraction.pdf_type == PDFType.scanned:
         - await update_job_status(job_id, "failed", error="PDF is scanned/image-based, cannot extract text")
         - return
