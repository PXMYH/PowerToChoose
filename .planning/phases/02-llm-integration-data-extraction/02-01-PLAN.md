       ```python
       import instructor
       import litellm
       from config import settings

       def get_llm_client():
           """Return an instructor-patched LiteLLM client for structured output extraction."""
           litellm.api_key = settings.OPENROUTER_API_KEY
           client = instructor.from_litellm(litellm.completion)
           return client
       ```
       This is intentionally thin. The extraction logic (prompt, retry, calling client) goes in Plan 02. This just provides the configured client factory.
