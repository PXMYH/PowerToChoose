# Quick Plan 260322-2gp: Migrate from aiosqlite to Turso (libSQL)

## Task
Replace aiosqlite with libsql for persistent cloud storage via Turso. Keep local SQLite as fallback when TURSO_DATABASE_URL is not set.

## Strategy
Based on research: libsql is sync-only, no row_factory. Keep all connection.py functions async by wrapping sync libsql calls in asyncio.to_thread(). This minimizes blast radius — callers don't change.

- Keep aiosqlite in test deps for existing test patterns if needed
